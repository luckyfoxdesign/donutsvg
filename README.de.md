# Piedog — SVG-Diagramm-Generator für Kreis- und Ringdiagramme

**Sprache:** [English](README.md) · [Русский](README.ru.md) · Deutsch · [Español](README.es.md)

Browser-Tool zur Erstellung von Kreis- und Ringdiagrammen mit Export als SVG oder PNG. Keine Anmeldung, keine Datenübertragung an einen Server.

[Live-Demo →](https://makemechart.com/chart/doughnut?old=true)

![Piedog preview](./public/img/workpreview.gif)

## Funktionen

- Bis zu 12 Segmente mit individuellen Farben und Werten
- Einstellbarer Außenradius, Innenradius und Abstand zwischen den Segmenten
- Export als SVG oder PNG
- SVG-Markup in die Zwischenablage kopieren — direkt in Figma oder Sketch einfügen

## Technologien

Svelte 5 · Svelte Material UI · Vite

## Lokal ausführen

```sh
docker compose up -d
```
