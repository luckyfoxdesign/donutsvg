# Piedog — Generador de gráficos SVG de sectores y anillos

**Idioma:** [English](README.md) · [Русский](README.ru.md) · [Deutsch](README.de.md) · Español

Herramienta de navegador para crear gráficos de sectores y de anillos y exportarlos como SVG o PNG. Sin registro, sin envío de datos a ningún servidor.

[Demo en vivo →](https://donutsvg.luckyfox.design/)

![Piedog preview](./public/img/workpreview.gif)

## Características

- Hasta 12 segmentos con colores y valores personalizados
- Radio exterior, radio interior y separación entre segmentos ajustables
- Exportar como SVG o PNG
- Copiar el marcado SVG al portapapeles — pegar directamente en Figma o Sketch

## Tecnologías

Svelte 5 · Svelte Material UI · Vite

## Ejecutar localmente

```sh
docker compose up -d
```
