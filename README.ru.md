# Piedog — Генератор SVG-диаграмм (круговых и кольцевых)

**Язык:** [English](README.md) · Русский · [Deutsch](README.de.md) · [Español](README.es.md)

Браузерный инструмент для создания круговых и кольцевых диаграмм с экспортом в SVG или PNG. Без регистрации, без отправки данных на сервер.

[Открыть демо →](https://donutsvg.luckyfox.design/)

![Piedog preview](./public/img/workpreview.gif)

## Возможности

- До 12 сегментов с произвольными цветами и значениями
- Настройка внешнего радиуса, внутреннего радиуса и зазора между сегментами
- Экспорт в SVG или PNG
- Копирование SVG-разметки в буфер — вставляйте напрямую в Figma или Sketch

## Технологии

Svelte 5 · Svelte Material UI · Vite

## Запуск локально

```sh
docker compose up -d
```
